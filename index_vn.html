Đồ án JAVA cuối kì: WebApp bán linh kiện máy tính

Tham khảo:
  • Github
  • Dữ liệu SQL
Chi tiết:
  • Cho phép khách hàng đăng ký/đăng nhập, xem chi tiết, đặt hàng và kiểm tra đơn hàng...
  • Cung cấp nhiều công cụ quản lý cho chủ cửa hàng (quản lý phân loại/sản phẩm/đơn hàng/thành viên/ nhân viên/ quyền… ).
Đạt được:
  • Kiến thức về Java EE.
  • Tự nghiên cứu và ứng dụng framework/engine (Spring MVC, Thymeleaf).
  • Dùng Maven để quản lý dependency.
  • Deploy dự án lên server Glassfish.

Quy mô: 3 người

Vai trò: Nhóm trưởng

Phần mềm:
  • Netbeans 8.2: Tạo dự án gốc, hỗ trợ gỡ lỗi, deploy dự án lên Glassfish.
  • Visual Studio Code: Công cụ hỗ trợ code.
  • SQL Server Management Studio: Quản lý CSDL SQL Server.
  • Github: Hỗ trợ làm việc nhóm, quản lý dự án.
Công nghệ:
  • Java EE (deploy trên server Glassfish)
  • CSDL MS SQL Server
  • Spring MVC
  • Java Persistent API
  • Thymeleaf
  • JQuery
  • Bootstrap
